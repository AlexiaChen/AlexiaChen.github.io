chatGPT临时代替版使用教程

背景

在chatGPT(GPT3.5) 如火如荼至今，这样类型的LLM工具助手，已经成为很多人不可或缺提升工作效率的工具。不论你是工程师，还是文职工作者可能都需要此工具。但是，使用chatGPT的成本对于个人用户来说，还是有些高昂，比如需要翻墙，OpenAI封IP和OpenAI账号，New Bing也是封IP， Bard又只能美国国家地区使用，只能使用美国节点才可以访问。所以，我开发部署了这个临时代替版，虽然没有chatGPT强大，但是基本的日常工作也够用了，这个是亲身体验，对我自己一个工程师来说，够用了，希望帮助到大家。

优势

  • 不需要翻墙
  • 国内其他的chatGPT镜像代理网站每天只能聊有限次数，限制比较大，而且也容易被封，网络经常出问题。因为调用chatGPT的API的转发服务器需要部署在海外节点
  • 价格非常便宜，chatGPT的API价格是1000个token 0.002美元，接近1.5分人民币左右。而临时代替版是1000个token 1.0分人民币，比chatGPT的API价格少了三分之一。(毕竟我要付电费，所以多少还是需要收费的。)
  • 对话效果质量接近chatGPT，比国内百度文心一言之流高很多。
  • 使用简单，只需几行配置即可

劣势

  • 中文对话效果一般，速度相对英文更慢。虽然这是劣势，但是也是优势，英文其实不是门槛，因为这个临时代替版的受众本来就是工程师，理工类行业的工作人群，不是公务员等行政类公文人群。做工程的，看英文资料是必须的，所以，这个劣势其实不是劣势，而是优势。

教程

文字版

  • 首先访问 https://web.chatboxapp.xyz/
  • 打开左下角的Setting配置
  • 选择OpenAI API这个AI provider
  • 填入需要找我申请的API key，申请后，看付款额度，充值对应的Token数量
  • API Host配置项填写入我提供的API URL地址。这个地址有可能会变更。所以需要加入社区群组，获取最新的API地址。API的地址形式是： https://be7c-222-172-244-12.ngrok-free.app
  • Model选项，选择gpt-3.5-turbo.
  • Temperature选项，选择0.7。 Temperature越低，生成的文本越保守(更基于事实)，越高，生成的文本越大胆（更有创造力）。需要看你平时的工作岗位属性来调整。
  • Max Tokens in Context设置为2048即可，这个不要过大，不然聊天回复会很慢。
  • Max Tokens per Reply设置为1024。
  • 点击Save保存配置。
  • 左下角点击New Chat按钮，开始一个新的聊天。（注意，初次聊天，回复速度可能比较慢，但是后面随着你的多轮对话，后端有相关的缓存机制，速度就会越来越快，趋于正常速度了）
